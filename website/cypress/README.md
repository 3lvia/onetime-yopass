# Cypress

## Getting Started

```bash
npx cypress run \
    --headless

npx cypress run \
    --headless \
    --spec="cypress/integration/create_secret.ts"

npx cypress run \
    --headless \
    --spec="cypress/integration/load_website.ts"

npx cypress run \
    --headless \
    --spec="cypress/integration/signin_testuser.ts"

npx cypress run \
    --headless \
    --spec="cypress/integration/upload_file.ts"
```
